# Supervised Machine Learning: Regression and Classification

Learning workspace for supervised machine learning concepts from the DeepLearning.AI / Stanford Machine Learning Specialization.

## Status

In progress. This repository is intentionally positioned as a learning repository, not a finished production project.

## Topics Covered

- Linear regression
- Logistic regression
- Classification fundamentals
- Gradient descent
- Cost functions
- Regularization
- Train/test evaluation
- Practical model interpretation

## Planned Structure

```text
notebooks/     Course notebooks and experiments
datasets/      Small public or synthetic datasets only
src/           Reusable helper scripts where useful
notes/         Concise technical summaries
```
